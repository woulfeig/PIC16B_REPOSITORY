We will fill our database using a function in a seperate Python file. From this file, we will import our function and as a result our database. This function requires us to use SQL keywords (written in all capital letters) in order to easily combine different sorts of data into one whole set.

from climate_database import query_climate_database
import inspect
print(inspect.getsource(query_climate_database))
def query_climate_database(db_file, country, year_begin, year_end, month):
    conn = sqlite3.connect(db_file)
    query = f'''
        SELECT s.NAME, s.LATITUDE, s.LONGITUDE, c.NAME AS Country, t.Year, t.Month, t.Temp
        FROM temperatures t
        LEFT JOIN stations s ON t.ID = s.id
        LEFT JOIN countries c ON t.ID LIKE c."FIPS 10-4" || '%'
        WHERE c.Name = '{country}' AND t.Month = {month} AND t.Year BETWEEN {year_begin} AND {year_end}

'''
    result_df = pd.read_sql_query(query, conn)
    conn.close()
    return result_df

Developing Visualizations from our Database

Now that we have all our data from our three sets organized, we can use them to answer interesting questions. Visualizations are a great way to display data as they are super versatile and easy to read. We will use Plotly to create our data sets. Please import the Plotly package below.

from plotly import express as px

Let’s get started with our first question.

How does the average yearly change in temperature vary within a given country?

The best way to tackle this question is through creating a geographic scatter function. This figure will appear as a map but will mark specific stations in a given country and share their temperature measurements. We will start by creating a function called tenperature_coefficient_plot(). Please follow the code written below to create our first visualization. We must be sure to include the necessary parameters and be as detailed as possible with our plots.

def temperature_coefficient_plot(db_file, country, year_begin, year_end, month, min_obs, **kwargs):
    df = query_climate_database(db_file, country, year_begin, year_end, month)
    df = df[df.groupby('NAME')['Year'].transform('count') >= min_obs]

    def temperature_average(data):
        X = data['Year']
        y = data['Temp']
        
        average = np.polyfit(X, y, deg=1)[0]
        
        return average

    coefs = df.groupby('NAME').apply(temperature_average).reset_index()
    coefs.columns = ['NAME', 'YearlyChange']

    df = pd.merge(df, coefs, on='NAME')

    fig = px.scatter_mapbox(df, 
                            lat='LATITUDE', 
                            lon='LONGITUDE',
                            color='YearlyChange',
                            hover_data = {'NAME' : True, 'YearlyChange': ':.3f'},
                            labels = {'YearlyChange': 'Estimated Yearly Increase (Celsius)'}, 
                            title = f"Estimates of Yearly Increase in Temperature in {pd.to_datetime(month, format='%m').month_name()} for stations in {country}, {year_begin}-{year_end}",
                            **kwargs)
    fig.update_layout(mapbox_style="open-street-map")
    return fig

Finally, let’s create one more visualization and learn more about temperature using our data sets. During a twenty-year period, which part of Italy experienced higher average temperatures during the month of August: North Italy, Central Italy, or South Italy?

As someone who has lived in Italy, I am very aware of the different lifestyles of the people in each of these three regions. Moreover, I would like to inquire whether climate has played a role in these cultural differences.

Once again, I have decided to use a scatterplot to answer my question. However, this scatterplot will differ from the one above as it analyzes the average temperature of each station rather than the change in temperature. Please see the code below.

def regional_difference_plot(db_file, country, year_begin, year_end, month, min_obs, **kwargs):
    df = query_climate_database(db_file, country, year_begin, year_end, month)
    df = df[df.groupby('NAME')['Temp'].transform('count') >= min_obs]

    def temperature_average(df):
        average = np.mean(df['Temp'])
        
        return average

    coefs = df.groupby('NAME').apply(temperature_average).reset_index()
    coefs.columns = ['NAME', 'Average Temperature']

    df = pd.merge(df, coefs, on='NAME')

    fig = px.scatter_mapbox(df, 
                            lat='LATITUDE', 
                            lon='LONGITUDE',
                            color='Average Temperature',
                            hover_data = {'NAME' : True, 'Average Temperature': ':.3f'},
                            labels = {'Average Temperature': 'Average Temperature (Celsius)'}, 
                            title = f"Temperatures in {pd.to_datetime(month, format='%m').month_name()} for stations in {country}, {year_begin}-{year_end}",
                            **kwargs)
    fig.update_layout(mapbox_style="open-street-map")
    return fig

Although we can apply this function to any country, month, or year, we will look at Italy in August over a twenty year period to develop our conclusion.
